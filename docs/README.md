## 🚀 快速开始

### 新用户入门路径

1. **了解项目** → 阅读 [01-项目总览文档](./01-项目总览文档.md)
2. **环境搭建** → 参考 [项目总览 - 快速开始](./01-项目总览文档.md#快速开始)
3. **功能学习** → 阅读 [02-审批流程文档](./02-审批流程文档.md) 和 [03-店铺详情页文档](./03-店铺详情页文档.md)
4. **API开发** → 参考 [04-后端API接口文档](./04-后端API接口文档.md)
5. **生产部署** → 按照 [05-生产环境部署文档](./05-生产环境部署文档.md) 操作

### 开发者路径

1. **克隆项目** → `git clone` 项目仓库
2. **安装依赖** → 参考 [项目总览 - 开发环境](./01-项目总览文档.md#开发环境)
3. **启动开发** → `bench start` 启动开发服务器
4. **前端开发** → `cd frontend && yarn dev` 启动Vue开发服务器
5. **API开发** → 参考 [API接口文档](./04-后端API接口文档.md)

### 运维人员路径

1. **服务器准备** → 参考 [部署文档 - 服务器要求](./05-生产环境部署文档.md#服务器要求)
2. **环境配置** → 按照 [部署文档 - 环境准备](./05-生产环境部署文档.md#环境准备) 操作
3. **应用部署** → 执行 [部署文档 - 部署应用](./05-生产环境部署文档.md#部署应用)
4. **监控维护** → 参考 [部署文档 - 监控和维护](./05-生产环境部署文档.md#监控和维护)
